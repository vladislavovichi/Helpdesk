# Продукт

## Что Такое Reliora

Reliora — внутренний helpdesk-продукт в Telegram для команд, которым нужен:

- быстрый операторский интерфейс;
- живой диалог с клиентом;
- понятный архив кейсов;
- спокойная аналитика и отчёты;
- отдельный backend без web-dashboard зависимости.

## Ключевые Capabilities

- intake по темам;
- queue workflow;
- active dialogue;
- macros, tags, notes;
- feedback и quality loop;
- archive browsing;
- historical ticket exports;
- management-friendly analytics exports.

## Product Language

Внешний интерфейс для операторов и администраторов остаётся:

- русским;
- коротким;
- спокойным;
- без бюрократического тона;
- без перегруженных экранов.

## Экспортная Линия

В проекте две отдельные группы отчётов:

- ticket reports — для конкретного кейса, handoff и внутреннего review;
- analytics reports — для операционного контроля, качества и SLA.
